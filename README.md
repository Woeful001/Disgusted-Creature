这个模组除了最初的几个物品，后面的代码基本都是用AI生成的，当然，这么丑的贴图和建模是自己做的。做完后没有太多喜悦，反而是一阵虚无。

我水平太差，建模也好，代码也好，做出来的始终达不到自己的预期，把想法砍了又砍。憋着一口气把自己最低预期给做完了。

为什么大量使用AI？一开始是因为对API等不熟悉，不知道该如何实现自己的想法，后面就纯粹是自暴自弃，没心思慢慢磨了。

该项目让我学到了什么？MC模组的基本结构，如何注册物品/实体等，增加最多的是建模水平（）

该项目的意义？我将利用余生研究我的模组（）  不得不说这个模组还是较为齐全地涉及模组开发的物品/实体等，用来供自己时不时研究模组开发挺好的

项目启示？我思考人和AI的关系，我想，当你是小白时，用AI完成项目后，无异于小学生看线代题的答案（略），是学不到什么东西的；
可若是至少掌握了基础知识（比如系统学习过java），这时用AI，就像是大学生看高数题的答案（有详细过程），哪里不懂，看一眼，思路和方法都能学到。

自我期望？稳扎稳打学习，逐渐摆脱对参考资料（AI）的依赖。
